# Приватный сервер Бравл Старс

Этот репозиторий содержит код для **приватного сервера Бравл Старс**
Сервер позволяет играть на неофициальной платформе, не подключаясь к официальным серверам Supercell

## Описание

Этот проект предоставляет возможность запустить свой собственный приватный сервер для игры **Brawl Stars**
Сервер поддерживает большинство основных функций игры, включая:

- Прокачка аккаунтов
- Возможность создавать акций в магазине
- Подключение к серверу через мобильные устройства (Пока что только Android)

## Структура репозитория

- `Messaging/` — логика обращения между клиента и сервера
- `Logic/` — основная логика сервера
- `Database/` — модели базы данных, круд
- `ContentUpdater/` — логика обновлений контента
